## Reg Description && Overview
register -> Description  
overview -> x0...x31 regs  
x0       -> zero constant  
x1       -> ret address  
x2       -> stack pointer sp  
x3       -> global pointer gp  
x5..x7   -> tmp regs  
x8..x9   -> saved regs  
x10..x11 -> function args regs  
x28..x31 -> tmp regs  
pc       -> program counter
